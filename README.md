# V-REP-UAV-simulation

Эксперименты функционирования БПЛА реализовывались с помощью встраиваемых скриптов V-REP, позволяющих программировать управление и поведение роботов напрямую в симуляторе, без установки дополнительного ПО. Используется встроенный редактор, поддерживающий скриптовый язык программирования Lua.

Основные характеристики частного случаю функционирования системы (эксперимента):

* квадратная область, по которой перемещаются агенты;
* 5 агентов + 1 диверсант;
* 10 флагов - 10 задач;
* область A – область старта/ область сдачи задания;
* потрачено минимум времени на выполнение задачи;
* выполнены все задачи;
* агенты – материальные точки.

Группа агентов начинает функционировать в области A. Целью группы является выполнение всех задач. Задачей является перемещение флага в область А , т.е. перемещение к точке, где находиться флаг, и дальнейшее перемещение в начало. Окончанием эксперимента является достижение цели.
